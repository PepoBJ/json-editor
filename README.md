## Keyboard Shortcuts

- `Ctrl/Cmd + Shift + F` - Format JSON
- `Ctrl/Cmd + Shift + M` - Minify JSON
- `Ctrl/Cmd + Shift + K` - Clear editor
- `Ctrl/Cmd + Shift + C` - Copy JSON
- `Ctrl/Cmd + Shift + D` - Compare JSON
